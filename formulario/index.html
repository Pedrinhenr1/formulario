A- Na tag input com atributo type="text", para que serve o atributo "placeholder"?
R: Placeholder serve para deixar uma pequena mensagem para ajudar o usuário a preencher o formulário


B- Na tag input com atributo type="radio" ou type="checkbox", para que serve o atributo "checked"?
R: Serve para deixar marcada ou alternada para um estado ligado/conectado


C- Na tag input para que serve o atributo "value"?
R: Serve para definir o valor a ser enviado por esse item


D- Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?
R: Tem que utilizar o atributo "name" e o "ID" assim irá conseguir marca 1 de cada vez


E- Para que serve a tag label? Para que serve o atributo "for" nessa tag?
R: Serve para deixar uma legenda para o usuário, E o atributo "for" na label serve para fazer uma ligação entre o label e o input


F- Na tag option para que serve o atributo "selected"?
R: Serve para mostrar o menu de opções


G- Nas tags de formulário para que serve o atributo "required"?
R: Serve para deixar alguma parte do formulário como ser obrigatório responder


H- Na tag form para que serve o atributo "action"?
R: Serve para definir um local para onde as informações irá ser mandadas


I- Na tag form para que serve o atributo "method"?
R: Serve para definir qual método HTTP para enviar os dados ( GET ou POST)


J- Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag form?
R: deve ser o POST


K- Qual valor deve ser informado no atributo "method" na tag form em um formulário que será usado para fazer upload de arquivos?
R:O formulário de upload é um formulário prático que permitr aos usuários enviar fotos, documentos ou qualquer outro tipo qualquer outro tipo de servidor e para criar um formulário de upload tem que estabelecer o valor de file (arquivo) para tag (input type=file)
